

Cuando el balón rebota,
el grito de gol se escucha,
la emoción se apodera,
el futbol es pasión.

El césped se mancha,
los pies se mojan,
el corazón se acelera,
el futbol es una locura.

Los goles se celebran,
el equipo se alienta,
el rival se respeta,
el futbol es una aventura.

Un partido se vive,
con ilusión y esfuerzo,
para ganar, para perder,
el futbol es un sentimiento.
